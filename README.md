# Simple web application scoreboard for fighting game tournaments

I have not created an executable, so run it directly from Visual Studio Code.

1. Clone repo
2. Open project from visual studio code
3. Run in terminal
```bash
   dotnet run
```
4. Open in browser: `http://localhost:5139`

## Urls
| URL | Description |
|---|---|
| `http://localhost:5139` | Scoreboard display |
| `http://localhost:5139/control.html` | Control panel |
